# 水果合併（Matter.js 範例）

簡單的物理合併遊戲範例，使用 Matter.js 實作。點擊畫面可丟出小水果，同等級碰撞即會合併為更高一級的水果（最多 10 級）。遊戲也會自動從頂部定時生成小水果。

快速玩法：

1. 用瀏覽器開啟 `index.html`。
2. 目前遊戲改為「預顯示下一顆要掉下的水果」，畫面上方會顯示 `預覽` 圖示。
   
	注意：遊戲一開始畫面為空，僅顯示預覽，請在畫面上按下或拖曳以放置第一顆水果。
3. 點擊畫面任意位置會把預覽中的水果從畫面頂端掉下到你點擊的 x 座標。

操作說明（觸控/拖曳）：

- 在手機或觸控板上，可按住並左右拖曳預覽以調整掉落位置；放開手指時水果會從頂端掉下。
- 在桌面上可用滑鼠按住並拖曳來定位，或直接點擊畫面快速丟下預覽水果。
4. 當兩顆相同等級且碰撞速度足夠時會合併為更高等級並得到分數。
5. 右上角可查看已達成的最高等級與目前分數，按下「重玩」重置遊戲。

提示：若要恢復自動生成或調整難度，可到 `main.js` 修改 `generateNext()` 的行為或加入生成計時器；合併速度門檻在 `MERGE_VELOCITY_THRESHOLD`。
